# AttendSecureX

A secure and intelligent campus attendance management system built with React Native. The platform combines QR-based attendance, location verification, administrative controls, and real-time tracking to create a reliable attendance ecosystem for educational institutions.

## Features

### Authentication & Access Control

* Student Login
* Admin Login
* First-Time Student Approval by Admin
* Secure Session Management
* Role-Based Access Control

### Attendance Management

* QR Code Based Attendance
* Instant Attendance Verification
* Attendance History Tracking
* Real-Time Attendance Status
* Course-Wise Attendance Records

### Location Verification

* GPS-Based Validation
* Campus Presence Verification
* Anti-Spoofing Controls
* Location-Based Attendance Approval

### Admin Dashboard

* Student Management
* Attendance Monitoring
* QR Session Creation
* Verification Settings
* Attendance Analytics
* Approval Requests Management

### Student Dashboard

* Mark Attendance
* View Attendance History
* Track Attendance Percentage
* Course Selection
* Attendance Status Overview

### Security Features

* QR Validation
* Location Verification
* Admin Approval Workflow
* Unauthorized Access Prevention
* Session Protection
